PythonMonkey: JavaScript/WASM Interop for Python — The alpha release of a new way to bring Python and JavaScript together, by way of embedding Mozilla SpiderMonkey into the Python VM. This post introduces the concept with some examples, an idea of where the project is headed, as well as some Colab demos. Despite using SpiderMonkey rather than V8, Node and npm compatibility is important to the project and more Node integration is on its roadmap. GitHub repo.

  • If you're using Node 14 on Vercel, this is your reminder that Vercel is discontinuing Node 14 support in just two weeks. Node 16 users get until next February to complete the switch – if you want to keep deploying your code, at least.

  • The Node.js team is celebrating 🐦 a 'long-term infrastructure transition' of the official Node site using Next.js and Vercel. Can we see any differences? Beyond some RSS feeds that broke overnight, no. But that's probably the point.

  • There are some anticipated Node.js security releases on August 8 to address seven security issues of varying severity.
