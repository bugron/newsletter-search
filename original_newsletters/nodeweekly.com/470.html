Bun v0.5 Released — Bun is another JavaScript runtime taking the world by storm and unlike Deno, some level of Node compatibility has been considered important from the start. v0.5 adds support for node:readline, workspaces, a node:dns shim, and network socket creation support such that more Node.js-based database libraries work out of the box.
