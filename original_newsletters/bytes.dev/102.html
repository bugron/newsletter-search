How does it work? fast-check provides a bunch of utilities for defining the ‚Äúproperties‚Äù of your code (things you assume will always be true).   3. Want less tests but with better coverage

Let‚Äôs get ready to bundle

Back in 2018, Airbnb did what lots of us have done when thinking about upgrading to a new version of Webpack ‚Äî give up.

Unfortunately for them, this was before The Great Build Tool Renaissance‚Ñ¢Ô∏è of the last couple years ‚Äî esbuild and Rollup weren‚Äôt around yet, Parcel wasn‚Äôt as robust as it is today, and Rome was still just a European city with lots of roads.

So let‚Äôs dive a little deeper into what makes Metro unique, why Airbnb decided to use it, and how it stacks up to some of the newer bundlers we have today.

There are two defining Metro features:

  1. On-demand bundle processing in development ‚Äî Metro only compiles JS bundles on the pages requested, as opposed to pre-compiling the entire project from the start (like Webpack).
  2. Multi-layered cache ‚Äî You can set up multiple caches to be used by Metro, and you get the flexibility to define the cache implementation you want.

For most projects, implementing these optimizations would probably feel like extreme hair-splitting (my favorite Winter X-Games event). import db from "./db";
import admin from "firebase-admin";

export default async function updateLessonComplete(req, res) {
  try {
    const { lessonId, completed } = req.body

    if (!lessonId) {
      throw new Error("lessonId is a required parameter");
    }

    if (!completed) {
