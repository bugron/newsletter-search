The React 19 Beta — The React team can say this beta is designed to get library developers prepared for an eventual React 19 release, but that hasn’t stopped everyone digging into all the new features anyway ;-) If you want to upgrade immediately, there’s a thorough upgrade guide, but otherwise, here are the highlights:

  • Full support for Custom Elements. – Custom Element support has always been a thorn in React's side due to how it treats props vs attributes, but finally there's a way!

  • All the latest React Server Components goodies. Natch.

  • Actions – You can now use async functions in transitions and get automatic optimistic updates. This leads to some opportunities for dramatic code simplification and smoother operation when working with forms (say hello to useActionState.)

  • use – Read a promise with use and React will Suspend until the promise resolves.

  • You can now render metadata tags (like title, link, and meta) in components natively.

  • Render async scripts anywhere in your component tree.

If you'd prefer to hear / see someone walking through the release post, I think ▶️ Theo did a pretty good job in 39 minutes.

It is, however, identical to 18.2 but adds warnings for deprecations, so if you’re not going to leap straight into 19, this is the more responsible path to take.
